🚗 Car Sales Data Analysis
Exploratory data analysis and visualization on a Car Sales dataset using Python.

📊 Visualizations Included

Bar Chart — Horsepower comparison across car brands
Box Plot — Distribution and outlier detection
Scatter Plot — Manufacturer vs Horsepower
Heatmap — Correlation between numerical features

📁 Dataset

File: Car_sales.csv
Key columns: Manufacturer, Horsepower, Sales, Price
